🌊 LLaMA 3 (8B)

Meta · апрель 2024

Создатель: Meta AI (Facebook). Одна из самых мощных открытых LLM с отличным пониманием инструкций.

⚡ Квантование: Q4_K_M
🎮 VRAM: 5.5 ГБ
🧠 RAM: 8+ ГБ
💾 Диск: 4.5 ГБ
# запуск через llama.cpp или Ollama
ollama run llama3:8b

# или llama.cpp:
./main -m llama3-8b-q4_K_M.gguf -n 256 --temp 0.7

Особенности: улучшенная мультиязычность, контекст 8k, отлично подходит для RAG и codegen. Рекомендуется RTX 3060 6GB+.

🌊 LLaMA 3 70B

Meta · энтерпрайз-класс

Создатель: Meta AI. Требует серьёзных ресурсов, но выдаёт уровень GPT-4 в многих задачах.

💾 Q4_K_M VRAM ~38 ГБ
🧠 RAM 64 ГБ + swap
📀 Диск: 35 ГБ
# llama.cpp с offload слоёв на GPU:
./main -m llama3-70b-q4.gguf -ngl 35 -c 4096
# или используйте exllamav2 для 2x24GB

Рекомендуется: 2×RTX 3090/4090 или A6000. Возможен CPU+GPU гибрид.
