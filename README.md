# Local_Neural_Network_Hardware_Requirements
# 🖥️ Local AI Hardware Requirements & Neural Profiles

Проект представляет собой интерактивный HTML-справочник в стиле Linux-терминала. Он содержит:

- **Главную таблицу** с требованиями к оборудованию (VRAM, RAM, диск, рекомендуемый GPU) для запуска популярных нейросетей локально.
- **Отдельные профили** для каждой модели с подробным описанием: создатель, особенности, команды запуска, а также встроенные видео с YouTube (обзоры, гайды по установке).

Все страницы выполнены в единой тёмно-синей и тускло-голубой цветовой гамме, стилизованной под командную строку Linux.

## 🚀 Демонстрация

Проект полностью статический (HTML/CSS/JS). Откройте файл `index.html` в любом современном браузере – никаких серверных зависимостей.

## ✨ Возможности

- **Терминальный интерфейс**: окно с заголовком, имитация командной строки, «мигающий курсор».
- **Главная таблица**:
  - 13+ популярных нейросетей (LLaMA 3, Mistral, Mixtral, Gemma 2, Phi‑3, Qwen 2.5, SDXL, Flux.1, Whisper, Bark, LLaVA и др.)
  - Квантование, требуемая VRAM, RAM, размер на диске, рекомендации по GPU.
- **Профили нейросетей**:
  - Создатель, краткое описание.
  - Спецификации (выделены в сетку).
  - Готовые команды запуска (Ollama, `llama.cpp`, Hugging Face Transformers, diffusers).
  - Два видео с YouTube (разбор + практический запуск).
- **Навигация**:
  - Из таблицы можно перейти в профиль по клику.
  - В каждом профиле есть кнопка «назад к таблице».
  - Отдельная кнопка в навигационной панели для возврата на главную.
- **Адаптивность**: таблица и карточки корректно отображаются на мобильных устройствах.
